# Prioritization Framework

## Core Principle
Maximize value and impact for minimal effort and cost.

## Process
1. **Create a spreadsheet** listing all initiatives
2. **Score each initiative** across key variables:
   - Impact
   - Effort
   - Cost
   - Time to market
   - Value
   - Mission alignment
3. **Sort and filter** to find highest and lowest alignment initiatives
4. **Prioritize** — top of the list = things we do, bottom = things we say no to

## Decision Rule
Even if impact and value are high, if effort is too high, don't pursue it now. Cut it.

## Re-evaluation
- Revisit deprioritized initiatives every ~6 months
- Re-score with updated context (new connections, capabilities, reduced costs)
- Set conditional triggers: "If KPIs look like X in 6 months, reconsider Y"

## Key Takeaway
Say no to low-alignment initiatives now. Stay focused on what matters today. Don't waste mental energy on things that aren't actionable yet.
