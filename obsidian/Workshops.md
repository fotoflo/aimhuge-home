# Workshops - AI Engineering in Production

The flagship [[Offerings|offering]] from AimHuge. Transform your team from AI-curious to AI-native.

## Format Options

- **2-day standard** - Immersive training + 24-hour hackathon
- **1-day focused** - For specific audiences
- **3-day extended** - Deep dive format

Every workshop is tailored to the organization's tech stack, goals, and team composition.

## Three Tracks

### [[Engineering Track]]
For developers & technical leads. Deep dive into Claude Code and Cursor, AI-assisted coding workflows, context management, production workflows.

### [[Product & PM Track]]
For product managers & designers. How AI changes product workflows, research, spec writing, prototyping, user testing.

### [[Executive Track]]
1:1 sessions for founders & C-suite on Claude Cowork and Claude Code, strategic decision-making.

## Standard Schedule

### Day 1 - Morning
- Skills assessment via Silent Meeting format
- Identify internal AI leaders
- Assess team baseline capability

### Day 1 - Afternoon
- Hands-on training with Claude Code & Cursor
- Build skills & commands library
- Learn plan mode, resume, and new commands

### Day 1 - Evening
- Form balanced hackathon teams
- Pitch project ideas and establish scope

### Overnight - 24-Hour Build Sprint
- Teams build real projects with AI-assisted development
- Mentoring available throughout
- Real deliverables: working software, not slides

### Day 2
- Final build sprint
- Project presentations & judging
- Team dinner & winner announcements

## Skills Taught

1. Claude Code & Cursor mastery
2. Plan mode, resume, and new commands
3. Building a skills & commands library
4. Voice-driven development
5. AI interviewing techniques for context
6. Prompt engineering for production code

## Key Methodologies

- **Silent Meeting** - Participants write about current usage, pain points, ambitions before discussion
- **Hands-on Building** - Everyone builds something real
- **Peer Learning** - Teams formed across skill levels
- **Practical Application** - Uses actual company challenges and codebase

## Expected Outcomes

- 2-5x faster development velocity
- Team-wide AI tool fluency
- Culture shift from AI-curious to AI-native
- Working software shipped during program
- Clear AI roadmap for organization
- Shared playbook of reusable prompts and workflows
