# WeGro Prioritization Framework

Strategic prioritization methodology for [[WeGro]] initiatives.

## Core Principle

Maximize value and impact for minimal effort and cost.

## Process

1. List all initiatives in a spreadsheet
2. Score each across: Impact, Effort, Cost, Time to Market, Value, Mission Alignment
3. Sort and filter for highest/lowest alignment
4. Top of list = do; bottom = say no

## Decision Rule

Even if impact and value are high — if effort is too high, don't pursue it now. Cut it.

## Re-evaluation Cycle

- Revisit deprioritized initiatives every ~6 months
- Re-score with updated context (new connections, capabilities, reduced costs)
- Set conditional triggers: "If KPIs look like X in 6 months, reconsider Y"

## Key Takeaway

Say no to low-alignment initiatives now. Stay focused. Don't waste mental energy on things that aren't actionable yet.
